# Naming Conventions

| Type     | Example                      |
| -------- | ---------------------------- |
| Function | function, <br> my_function   |
| Variable | x, var, <br> my_var          |
| Class    | Model, <br> MyClass          |
| Method   | class_method, <br> method    |
| Constant | CONSTANT, <br> MY_CONSTANT   |
| Module   | module.py, <br> my_module.py |
| Package  | package, <br> mypackage      |
